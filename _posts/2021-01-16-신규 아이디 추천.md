# 신규 아이디 추천

정규식을 이용한다면 쉽게 풀 수 있습니다. 조만간 정규식에 대한 포스팅도 할 예정이니 혹시나 정규식을 모르시면 독학하시거나 그냥 넘어가셔야겠습니다.

## 코드

```python
import re   
def solution(new_id):
    new_id = re.sub('[^a-z0-9-_.]', '', new_id.lower())
    new_id = re.sub('\.+', '.', new_id).strip('.')
    if not new_id :
        new_id = 'a'
    if len(new_id) >= 16:
        new_id = new_id[:15]
    new_id = new_id.rstrip('.')
    if len(new_id) <= 2:
        last_char = new_id[-1]
        while len(new_id) < 3:
            new_id += last_char    
    return new_id
```
