# 스킬 트리

```python
import re
def solution(skill, skill_trees):
    answer = 0
    for skill_tree in skill_trees :
        tmp = re.sub('[^'+skill+']', '', skill_tree)
        answer += 1 if tmp == skill[:len(tmp)] else 0
    return answer
```

---









## 풀이

각 `skill_trees` 원소에서 `skill`과 일치하는 알파벳을 제외하고 싹 날린다음에 `skill` 순서와 맞는지 검사하면 됩니다.
