# 다음 큰 숫자

```python
def solution(n = 78):
    b = bin(n)[2:]
    index = b.rfind('01')
    if index != -1 :
        b = b[:index] +'10' +''.join(b[index+2:][::-1])
    else :
        b = b[0]+'0'+ ''.join(b[1:][::-1])
    return int(b, 2)
```
