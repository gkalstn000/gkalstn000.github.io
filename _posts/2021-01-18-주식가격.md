# 주식가격

## 코드

```python
def solution(prices):
    n = len(prices)
    answer = [0]* n
    for i in range(n) :
        for j in range(i+1, n) :
            answer[i] += 1
            if prices[i] > prices[j] :
                break
    return answer
```

---







## 풀이

거품정렬 알고리즘을 이용했습니다. 계속 시간초를 늘려주다가 주식이 떨어지는 시점에 `break.`
