```python
def divisor(n) :
    for i in range(2, int(n**0.5)+1) :
        if n%i == 0 and n//i <= 10000000 : return n//i
    return 1 if n > 1 else 0

def solution(begin, end):
    return [divisor(x) for x in range(begin, end+1)]
```
