# Target

**BLDS**

* 1 : 당뇨병 `if BLDS >= 127`
* 0 : 정상인 `if BLDS < 127`  

Target은 결측치를 채워 넣을 수 없으니 결측치 `drop` 한다.  
12.3 : 1 비율의 불균형 데이터


```python
df.loc[ df['BLDS'] < 127, 'BLDS'] = 0
df.loc[ df['BLDS'] >= 127, 'BLDS'] = 1
df = df.dropna(subset = ['BLDS'], how = 'any', axis=0)
```
