```python
def solution(strs, t):
    DP = [0] * ((n := len(t)) + 1)
    for i in range(1, n+1) :
        DP[i] = min((DP[k] + 1 for k in range(max(0, i - 5), i) if t[k:i] in strs), default = 1e6)
    return DP[-1] if DP[-1] < 1e6 else -1
```

---









## DP의 뜻

```
DP[i] : t[i]를 만들기 위해 사용되는 strs의 수
```
