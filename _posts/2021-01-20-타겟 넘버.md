# 타겟 넘버

```python
from itertools import product
def solution(numbers, target):
    l = [(x, -x) for x in numbers]
    s = list(map(sum, product(*l)))
    return s.count(target)
```

---









## `product`

`product` 함수는 카디션곱을 해준다.

```python
a = ['ab', '12', 'AB']
list(product(*a))
```

```
[('a', '1', 'A'),
 ('a', '1', 'B'),
 ('a', '2', 'A'),
 ('a', '2', 'B'),
 ('b', '1', 'A'),
 ('b', '1', 'B'),
 ('b', '2', 'A'),
 ('b', '2', 'B')]
```

## 풀이

> 각 숫자별 `+/-` 기호를 넣은 경우의 수를 만들어 `map(sum, 카디션곱)` 리스트를 만든다
>
> `target` 의 개수를 센다
