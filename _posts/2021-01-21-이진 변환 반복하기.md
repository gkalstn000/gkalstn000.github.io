# 이진 변환 반복하기

```python
import re
def solution(s = "110010101001"	):
    zeros = 0
    t = 0
    while s != '1' :
        zeros += s.count('0')
        t += 1
        s = bin(len(re.sub('0', '', s)))[2:]
    return [t, zeros]
```
