# 시저 암호

아스키코드와 `isupper()` & `islower()` 를 이용해 정말 간결하고 깔끔하게 잘짠 코드입니다.

```python
def caesar(s, n):
    s = list(s)
    for i in range(len(s)):
        if s[i].isupper():
            s[i]=chr((ord(s[i])-ord('A')+ n)%26+ord('A'))
        elif s[i].islower():
            s[i]=chr((ord(s[i])-ord('a')+ n)%26+ord('a'))
    return "".join(s)
```
