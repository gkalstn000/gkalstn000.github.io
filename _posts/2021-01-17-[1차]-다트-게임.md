# [1차] 다트 게임

다트 각 구간별로 점수계산만 잘해주면 된다.

```python
def solution(dartResult):
    answer = ['0']
    space = {'S' : '**1', 'D' : '**2', 'T' : '**3'}
    tmp = ''
    for ch in dartResult :
        if ch == '*' :
            answer[-1] += '*2'
            answer[-2] += '*2'
        elif ch == '#' :
            answer[-1] = '-'+answer[-1]
        elif ch in ['S', 'D', 'T'] :
            tmp += space[ch]
            answer.append(tmp)
            tmp = ''
        else :
            tmp += ch
    return eval('+'.join(answer[1:]))
```
