# 쿼드압축 후 개수 세기

[쿼드트리 설명](https://m.blog.naver.com/PostView.nhn?blogId=qkfkf123&logNo=60179360083&proxyReferer=https:%2F%2Fwww.google.com%2F){: target="_blank"}

```python
import numpy as np
def solution(arr):
    arr = np.array(arr)
    answer = [0, 0]
    def quad(arr) :
        n = len(arr)
        if np.sum(arr) == 0 or np.sum(arr) == n**2:
            answer[arr[0][0]] += 1
            return
        quad(arr[:n//2, :n//2])
        quad(arr[:n//2, n//2:])
        quad(arr[n//2:, :n//2])
        quad(arr[n//2:, n//2:])
    quad(arr)
    return answer
```
